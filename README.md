🔄 Hợp nhất đa luồng (Cross-Log Correlation): Tự động "đối chiếu" Traffic, Threat và URL log. Nhìn phát biết ngay IP nào đang khai thác lỗ hổng gì và truy cập domain nào trong 1 truy vấn duy nhất.

📊 Dựng lại hiện trường (Visual Forensics): Lập bản đồ Kill-chain bằng biểu đồ Sankey 5 tầng và phát hiện Scanning/C2 Server bằng Force-Directed Graph.

⚡ Điều tra chớp nhoáng (Pivot & Triage): Cơ chế "Click-to-Filter". Bấm vào 1 IP/App bất kỳ, toàn bộ Dashboard và Event Timeline lập tức cô lập mục tiêu để truy vết.

🛠 Lõi xử lý siêu tốc: Sử dụng DuckDB (Embedded OLAP) đọc file nén Parquet, xử lý Big Data với độ trễ gần như bằng 0.
